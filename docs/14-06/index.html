Encargo lunes 14 de junio

Art Installation

  1. Análisis e investigación del referente.

El pabellón temporal ICD/ITKE 2010 es una estructura flexible hecha completamente de tiras extremadamente finas de madera contrachapadas, dobladas elásticamente, punto que como grupo nos llama mucho la atención. Creemos que es muy interesante como con tablas de madera planas se puede llegar a formar una estructura como este pabellón, es por eso que decidimos tratar de seguir esta técnica simplificando un poco con el fin de facilitar la construcción de la estructura, y como primer acercamiento estuvimos cambiando algunos parámetros de construcción para llegar a nuestra primera propuesta. Así utilizamos tiras de madera más delgadas que parten de una base entrelazada y terminan como una especie de “palmera”. El desafío en este caso deriva a cómo lograr que estas finas maderas se mantienen dobladas sin ceder ante la fuerza elástica, para ayudar a resolver esto utilizaremos vidrio (o algún plástico) además de cumplir con una función de refugio. Especialmente nos interesa el concepto de cómo llegar a una estructura tan grande y espaciosa en base a componentes pequeños y delgados.
